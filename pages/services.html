OLAC Infrastructure Services

The following services support the operation of OLAC.

Archive Registration

A service for validating and registering OLAC archives.
Freestanding Metadata

A service for validating and formatting an OLAC metadata record.
OLAC Aggregator

A service that provides a repository containing the records from all other registered OLAC repositories, incorporating an OLAC-DC crosswalk and a query function.
Viser

A virtual service which allows language resource sites to post services based on OLAC metadata without having to harvest metadata or develop a user interface.

OLAC Utility services

The following services support the developers of OLAC archives, and are provided on the OLAC Tools page.

  • Plain text search on all harvested metadata.
  • Resolve an identifier to a record.
  • Survey of metadata element usage.
